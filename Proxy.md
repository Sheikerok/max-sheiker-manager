## 🛒 Часть 1: Покупка прокси на Proxy-Market

### 1️⃣ Регистрация на сервисе

Перейдите по ссылке для регистрации на сервисе Proxy-Market:
👉 [Зарегистрироваться](https://ru.dashboard.proxy.market/?ref=cubepromocode12190352)

Заполните форму регистрации или войдите через удобный для вас способ (Google, Telegram и т.д.).

### 2️⃣ Переход к покупке серверных прокси

После входа в личный кабинет перейдите в раздел покупки:
**Серверные → Покупка** 

Или по прямой ссылке 👉 [Серверные прокси](https://ru.dashboard.proxy.market/proxy/datacenter/buying)

### 3️⃣ Выбор типа прокси

На странице покупки вы увидите два основных типа:

| Тип | Описание | Скорость |
|------|------|------|
| **IPv4** | Выделенный IP только для вас | 🚀 Высокая |
| **IPv4 Shared** | IP делится с другими пользователями | 🐢 Ниже |

✅ Для работы с MAX Sheiker Manager выбирайте **IPv4** для максимальной производительности.

### 4️⃣ Выбор срока аренды

Выберите период, на который хотите арендовать прокси:
- 1 месяц
- 3 месяца
- 6 месяцев
- 1 год

💡 *Чем дольше срок, тем выгоднее цена за месяц.*

### 5️⃣ Выбор страны и тарифа

Выберите страну **Россия**. Доступны три тарифа:

| Тариф | Скорость | Особенности |

## 💻 Часть 2: Добавление прокси в MAX Sheiker Manager

### 1️⃣ Открытие вкладки "Прокси"

Запустите MAX Sheiker Manager и перейдите на вкладку **"Прокси"** (она находится самой первой среди вкладок с функционалом).

### 2️⃣ Активация использования прокси

Установите чекбокс **"Использовать прокси"**.  
Как только вы это сделаете, интерфейс управления прокси станет активным — появятся кнопки и таблица для списка прокси.

### 3️⃣ Добавление нового прокси

🟧🟧🟧 ИЗ ФАЙЛА 🟧🟧🟧

Нажмите кнопку **"Импорт из файла"**.  
Перед вами откроется окошко для выбора текстового файла с прокси. Здесь нужно выбрать наш ранее скачанный файл.

| **IP адрес** | `45.15.72.199` | IP-адрес прокси-сервера |

### 4️⃣ Автоматическая проверка прокси

После добавления прокси программа **автоматически проверит его работоспособность**:

- 🟢 **Зеленый цвет** — прокси рабочее, можно использовать
- 🔴 **Красный цвет** — прокси не работает

Если прокси нерабочее, информация об ошибке будет отображена в логе программы. Это поможет понять причину: неверные данные, недоступность сервера или проблемы с типом подключения.

Рядом с прокси также отобразится страна (ГЕО), определенная по его IP. Это полезно для контроля геолокации ваших аккаунтов.

### 5️⃣ Управление прокси (дополнительно)

После добавления прокси вы можете:

| Действие | Описание |
|------|------|
| **Удалить** | Выделите прокси в списке и нажмите кнопку "Удалить" |
| **Проверить прокси** | Выделите прокси и нажмите кнопку "Проверить прокси" для ручной повторной проверки |

### 6️⃣ Начало работы

После успешного добавления и проверки прокси можно приступать к работе с другими вкладками программы. Все запросы от ваших аккаунтов будут проходить через выбранные прокси.

**Вопрос:** Сколько прокси нужно покупать?  
**Ответ:** Рекомендуется использовать один прокси на 1-3 аккаунта для равномерного распределения нагрузки.

**Вопрос:** Что делать, если прокси не работает?  
**Ответ:** Проверьте правильность введенных данных, попробуйте изменить тип прокси (http/socks) или обратитесь в поддержку Proxy-Market.

**Ответ:** Крайне не рекомендуется — они нестабильны, медленны и могут быть уже забанены в мессенджере.

**Ответ:** Да, разработчик тестировал программу именно на этом тарифе — работа стабильная.
