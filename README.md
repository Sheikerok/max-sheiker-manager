# MAX Sheiker Manager — Professional Account Manager

Мощный профессиональный менеджер аккаунтов для работы с мессенджером MAX. Программа предоставляет комплексный инструментарий для автоматизации работы с аккаунтами, парсинга чатов, рассылки сообщений и управления профилями.

## Основные возможности

- 1️⃣ **Управление множеством аккаунтов** — работа с неограниченным количеством аккаунтов, проверка их валидности.
- 2️⃣ **Парсинг чатов** — автоматический сбор участников из групп и каналов.
- 3️⃣ **Парсер контактов** — автоматический сбор контактов в текстовый файл в виде ID.
- 4️⃣ **Вступление в чаты** — массовое присоединение к чатам по пригласительным ссылкам.
- 5️⃣ **Инвайтинг по ID** — приглашение пользователей в группы и чаты по ID (по одному или пачками).
- 6️⃣ **Рассылка сообщений по ID** — массовая отправка сообщений по ID пользователей.
- 7️⃣ **Рассылка по чатам** — массовая отправка сообщений по чатам (либо по указанным ссылкам, либо по уже вступившим).
- 8️⃣ **Управление профилями** — настройка имен, фамилий, био, фото и 2FA для выбранных аккаунтов.
- 9️⃣ **Прогрев** — автоматически аккаунты выполняют рандомные действия в мессенджере.
- 1️⃣0️⃣ **Чистка чатов** — массовый выход из чатов и каналов.
- 1️⃣1️⃣ **Чек номеров** — находит аккаунты MAX по номеру телефона и получает их ID.
- 1️⃣2️⃣ **Глобальный лог** — пишет все действия, выполняемые программой с детальной информацией.
- 1️⃣3️⃣ **Автоматическое обнаружение** - программа автоматически находит аккаунты в папке cache.
- 1️⃣4️⃣ **Прокси** — гибкая настройка и проверка прокси для каждого аккаунта.

## Установка и запуск
1. Скачайте последнюю версию программы
2. Распакуйте архив в удобную папку
3. Запустите `MAX_Sheiker_Manager.exe`

## 🌐 Вкладка "Прокси"

### Функционал
- Централизованное управление прокси для всех аккаунтов.
- Поддержка HTTP, HTTPS, SOCKS4, SOCKS5.
- Автоматическая проверка работоспособности и определение GEO прокси.
- Подробный лог ошибок подключения.

### Как использовать
1.  **Активация**: Установите чекбокс **"Использовать прокси"**. Интерфейс управления станет активным.
2.  **Добавление прокси**: Нажмите кнопку **"➕ Добавить"**.
3.  **Заполнение формы**: В открывшемся окне укажите:
    *   **IP адрес**
    *   **Порт**
    *   **Логин** (если требуется)
    *   **Пароль** (если требуется)
    *   **Тип**: Выберите из списка (http/socks4/socks5)
4.  **Автоматическая проверка**: После добавления прокси будет автоматически проверено.
    *   🟢 **Зеленый**: Прокси рабочее.
    *   🔴 **Красный**: Прокси не работает. Информация об ошибке будет отображена в логе.
    *   Рядом с прокси будет указана страна (ГЕО), определенная по его IP.
5.  **Управление**: Выделите прокси в списке и используйте кнопки **"Удалить"** или **"Проверить прокси"** для ручной проверки.

## 👥 Вкладка "Аккаунты"

### Функционал
- Добавление новых аккаунтов.
- Управление существующими аккаунтами.
- Подключение/отключение от аккаунтов.
- Автоматическое обнаружение аккаунтов из папки cache.
- Возможность открыть аккаунт во встроенном браузере.
- **Новая функция: Проверка валидности выделенных аккаунтов.**

### Как использовать
1.  **Добавить аккаунт**: Нажмите "➕ Добавить аккаунт", введите номер телефона в формате `+79123456789`
2.  **Авторизация**: Программа запросит QR-код подтверждения из мессенджера MAX.
3.  **Подтверждение**: Отсканируйте QR-код из консоли.
4.  **Подключение**: Выберите аккаунт из списка и нажмите "🔗 Подключить".
5.  **Обновление**: Нажмите "🔄 Обновить" для сканирования папки cache на новые аккаунты.
6.  **Удаление**: Выберите аккаунт и нажмите "🗑️ Удалить аккаунт" для полного удаления.
7.  **Проверка валидности**:
    *   Выделите галочками нужные аккаунты.
    *   Нажмите кнопку "✅ Проверить валидность".
    *   Программа подключится к каждому аккаунту и попытается отправить тестовое сообщение.
    *   **Результат:**
        *   🟢 **Зеленый**: Аккаунт подключился, сообщение отправилось — полностью валидный.
        *   🟡 **Желтый**: Аккаунт подключился, но сообщение не отправилось — есть ограничения.
        *   🔴 **Красный**: Аккаунт не подключился — заблокирован или невалидный.

### 💡 Советы
- Аккаунты автоматически обнаруживаются в папке `cache/`.
- Для работы требуется стабильное интернет-соединение.
- Рекомендуется использовать номера с российскими кодами (+7).
- Теперь в таблице аккаунтов сохраняются и отображаются Имя и Фамилия владельца.

## ⚙️ Настройки и лицензирование

### Лицензирование
Программа использует систему онлайн-лицензирования:
- При первом запуске запрашивается никнейм и лицензионный ключ
- Данные проверяются онлайн

### Обработка ошибок
- Автоматическое переподключение при обрывах связи
- Детальные сообщения об ошибках
- Защита от частых запросов

## 🔒 Безопасность

### Хранение данных
- Сессии аккаунтов хранятся в папке `cache/`
- Данные аккаунтов шифруются средствами MAX API
- Лицензионная информация защищена

### Рекомендации по безопасности
- Регулярно обновляйте программу
- Используйте надежные пароли для аккаунтов
- Храните backup папки `cache/`
